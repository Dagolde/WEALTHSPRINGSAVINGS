## 💡 Common Tasks

### Create a New Admin
1. Go to Users
2. Click "Create User"
3. Fill details
4. Set Role to "Admin" ⭐
5. Click "Create User"

### Create a Regular User
1. Go to Users
2. Click "Create User"
3. Fill details
4. Set Role to "User"
5. Click "Create User"

### Promote User to Admin
1. Go to Users
2. Find user
3. Click "Edit"
4. Change Role to "Admin" ⭐
5. Save Changes

### Update Your Profile
1. Click "My Profile" ⭐
2. Edit your details
3. Click "Update Profile"

### Change Your Password
1. Click "My Profile" ⭐
2. Scroll to "Change Password"
3. Enter passwords
4. Click "Change Password"

### Approve KYC
1. Go to KYC Approvals
2. Review submission
3. Click "Approve"

### Approve Withdrawal
1. Go to Withdrawals
2. Review request
3. Click "Approve"
